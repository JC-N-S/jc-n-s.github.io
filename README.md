## 如何发布一篇新文章？

**完全不需要安装任何软件**，在浏览器里就能完成。跟着下面 3 步走：

### 第 1 步：复制模板

1. 打开 `articles/` 文件夹
2. 点击 `template.html` 文件
3. 点击右上角的"复制"按钮（或 Raw 模式下全选复制）
4. 回到 `articles/` 文件夹，点击"Add file" → "Create new file"
5. 文件名填 `你的文章名.html`（用英文或拼音，不要中文）
6. 把模板内容粘贴进去

### 第 2 步：修改内容

在粘贴的代码里，找到所有标有 `改这里` 的注释，修改以下内容：

- **文章标题**（两处：`<title>` 和 `<h1>` 里面）
- **文章描述**（`<meta name="description">`）
- **关键词**（`<meta name="keywords">`）
- **所属频道**（面包屑导航里的链接和名称）
- **频道标签颜色类名**（`tag-game` 改成对应频道的类名，见下表）
- **发布日期**
- **正文内容**（在 `<div class="article-body">` 里面写）

**频道标签类名对照表：**

| 频道 | 类名 |
|------|------|
| 游戏攻略 | `tag-game` |
| 知识课程 | `tag-knowledge` |
| 生活分享 | `tag-life` |
| 影视推荐 | `tag-movie` |
| 软件工具 | `tag-software` |
| 资源导航 | `tag-resources` |
| 科技数码 | `tag-tech` |
| 读书写作 | `tag-reading` |

### 第 3 步：更新频道页和首页

文章发布后，还需要把它加到对应的频道列表页里，这样读者才能看到：

1. 打开对应频道的文件（如 `channels/game.html`）
2. 找到 `article-list` 区域
3. 在第一个文章卡片后面，复制一份空白的文章卡片结构，填上你的文章信息
4. 如果想让这篇文章出现在首页"最新文章"区，打开 `index.html` 同样操作

## 关于广告

所有广告位代码里都已经预留好了，但默认是**隐藏**的（`style="display:none"`）。

等你网站有了稳定的日访问量（建议 > 100/天），再申请 Google AdSense。通过审核后，把广告位代码里的 `style="display:none"` 删掉就行。

广告位位置：
- 文章页：顶部（小横幅）、中段（2-3段后）、底部
- 频道列表页：每5-6篇文章之间
- 首页：最新文章和频道精选之间

## 如何部署到 GitHub Pages？

### 第一次部署

1. 注册 GitHub 账号（github.com）
2. 创建新仓库，名称填 `你的用户名.github.io`
3. 把本文件夹里所有文件上传到仓库
4. 进入仓库 Settings → Pages → Source 选 `main` 分支 → Save
5. 等 1-2 分钟，网站就在 `https://你的用户名.github.io` 上线了

### 绑定自己的域名

1. 先买个域名（Namecheap 或阿里云，约 70-100 元/年）
2. 编辑 `CNAME` 文件，把 `你的域名.com` 改成你的真实域名
3. 在域名管理后台添加 GitHub Pages 的 DNS 记录
4. GitHub 仓库 Settings → Pages → Custom domain 填写你的域名
